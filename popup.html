Welcome to AI Cost & Usage Tracker

Everything below stays on your device. The extension watches network traffic on supported AI sites in your own browser session and counts tokens and cost locally.

  • No account required for the extension itself. You only need to be signed in to the AI platforms you use (Claude, ChatGPT, Gemini, Mistral) so the extension can see your traffic.
  • No telemetry, no servers. The only optional outbound call is to api.anthropic.com for more accurate Claude token counting, and only if you opt in with your own API key.
  • Find settings under the Tools and Insights tabs: budget alerts, region for carbon estimates, model aliases, retention, and data export.
